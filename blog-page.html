'var', 'let' and 'const' all are used to define variables.
var: var is a function scope variable. Using var, they are accessed within the declared function.
let: let is a block scope variable. Using let, they are accessed within the block scope.
const: const is a block scope variable. After assigning a value to const, it cannot be changed or modified.

NULL: Null means empty, no value, or not existing.
Undefined: Undefined means a variable is declared, but its value is not assigned.

REST API: An interface between frontend and backend. It acts as a middleman between the two, facilitating data transfer from the API to the frontend.
